Mandatory file requirements

For reliable biomarker analysis, your file must include:

  • one structured table with a clear header row
  • one label column such as `label`, `risk_label`, `class`, or `target`
  • numeric feature columns such as reading speed, error count, hesitation count, gaze values, or timing values
  • one sample per row
